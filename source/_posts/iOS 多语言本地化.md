#### `插件`

`AMLocalizedStringBuilder`

快捷键 `ctrl+f`

把`Localizable.strings`生成为`object-c`类。

使用

```objectivec
#import "AMLocalizedString.h"

- (void)viewDidLoad {
    [super viewDidLoad];
    R_String.am_Chinese;
}
```
