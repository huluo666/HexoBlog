# Hexo博客
```
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo clean
hexo server
```



> WARNING
>
> 如果你没有 pug 以及 stylus 的渲染器，请下载安装：
>
> npm install hexo-renderer-pug hexo-renderer-stylus --save 或者
> yarn add hexo-renderer-pug hexo-renderer-stylus



```
$ npm install hexo-generator-search --save
```
